# Hybrid Fitness OS

Persönliches Fitness-Dashboard für Hybrid-Athleten.

## Was ist das?
Ein cloudbasiertes Dashboard für Hybrid-Athleten das:
- Daten von Garmin, Training Peaks und Heavy App zusammenführt
- Vergangene Trainings analysiert und bewertet
- Zukünftige Trainings plant und visualisiert
- Basierend auf allen Daten automatisch neue Trainingspläne erstellt

## Tech Stack
- Python (Backend)
- PostgreSQL (Datenbank)
- Streamlit (Dashboard)
- Railway (Cloud Hosting)
