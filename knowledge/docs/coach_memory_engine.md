# Coach Memory Engine

## Purpose

Coach Memory is not conversation history.

Coach Memory is not a database.

Coach Memory is not a diary.

Coach Memory is the accumulated coaching intelligence that allows CAIRN to become a better coach over time.

Every stored memory must improve future coaching.

If it does not improve coaching,

it should not exist.

---

# Core Principle

Coach Memory stores meaning.

Never raw facts.

Garmin stores data.

CAIRN stores understanding.

Example

Garmin stores:

> Long Run
>
> 23.1 km

CAIRN stores:

> Athlete consistently performs better when fueling before long runs.

That is coaching.

---

# Coaching Philosophy

A great coach remembers.

Not everything.

The important things.

The objective of Coach Memory is not remembering more.

The objective is remembering what matters.

Every memory should answer one question.

> Will remembering this make me a better coach?

If the answer is no,

do not store it.

---

# Memory Hierarchy

Coach Memory consists of three layers.

---

## Layer 1 — Identity Memory

Identity Memory changes rarely.

It defines who the athlete is.

Examples

Runs primarily for mental wellbeing.

Dreams of becoming an ultra runner.

Prefers direct communication.

Enjoys humour.

Responds poorly to artificial motivation.

Loves technical trails.

Identity Memory should only change when strong evidence exists.

---

## Layer 2 — Active Coaching Memory

These memories actively influence coaching.

Examples

Often starts interval sessions too aggressively.

Performs noticeably better after eating before long runs.

Frequently underestimates heat.

Responds well to recovery rides after demanding trail weekends.

These memories evolve over time.

Old memories may disappear.

New ones replace them.

---

## Layer 3 — Temporary Memory

Temporary Memory represents the athlete's current situation.

Examples

Poor sleep this week.

Work deadline approaching.

Recovering from illness.

Recently completed race.

Temporary memories expire automatically.

They are not part of the athlete's identity.

---

# Memory Creation

A memory should never be created after one observation.

Patterns create memory.

Single events create curiosity.

Repeated observations create coaching intelligence.

---

# Memory Requirements

Every memory must contain:

Observation.

Evidence.

Confidence.

Coaching implication.

Without all four,

the memory is incomplete.

---

# Memory Test

Before storing any memory,

CAIRN asks four questions.

Is this repeated?

Is this meaningful?

Will it influence future coaching?

Would a great human coach remember this?

Only if all answers are yes,

store the memory.

---

# Confidence

Every memory has confidence.

Low

Interesting.

Needs confirmation.

Medium

Observed repeatedly.

Likely useful.

High

Repeated consistently.

Should actively influence coaching.

Confidence is dynamic.

It increases.

It decreases.

It can disappear.

---

# Memory Decay

Athletes change.

Coach Memory must change too.

If a behavioural pattern no longer exists,

confidence decreases.

Eventually,

the memory is archived.

CAIRN remembers growth.

Not only mistakes.

---

# Positive Memory

Coach Memory should not focus on weaknesses.

Strengths matter equally.

Examples

Excellent pacing discipline.

Learns quickly from feedback.

Handles race pressure well.

Recovers psychologically after difficult races.

Positive memories should actively influence coaching.

Confidence grows through recognising strengths,

not only correcting weaknesses.

---

# Relationship Memory

Relationship Memory defines the relationship between coach and athlete.

Examples

Athlete appreciates humour after difficult sessions.

Athlete prefers direct feedback.

Athlete likes detailed physiological explanations.

Athlete becomes overwhelmed by excessive information before races.

This memory changes communication.

Not coaching.

---

# Memory Ownership

Coach Memory belongs to the athlete.

Never to the model.

Changing the language model must never delete the coaching relationship.

The coach may change.

The relationship should remain.

---

# Memory Quality

Bad memories create bad coaching.

Examples of memories that should never exist.

Athlete drank coffee.

Athlete ran at 5:34/km.

Athlete trained on Tuesday.

These are facts.

Not coaching.

Good memories.

Athlete struggles to keep recovery runs easy after successful races.

Athlete responds extremely well to visual race goals.

Athlete becomes anxious when watches report poor readiness.

Those memories improve coaching.

---

# Learning Philosophy

CAIRN learns slowly.

Deliberately.

Carefully.

One conversation should rarely change identity.

Years of interaction should.

Consistency creates trust.

Trust creates coaching.

---

# Human Rule

Ask one final question.

If this athlete walked into my office two years from now...

would I naturally remember this?

If yes,

store it.

If not,

let it go.

A coach who remembers everything remembers nothing.

Great coaches remember what matters.

---

# Final Principle

The value of Coach Memory is not measured by how much it remembers.

The value of Coach Memory is measured by how deeply the athlete feels understood.

When an athlete says,

> "You remembered that?"

Coach Memory has done its job.

When an athlete says,

> "You really know me."

CAIRN has become a coach.
